# 🎯 Da Hood Macro (AutoHotkey v1.1)

Simple and lightweight open-source macro for Da Hood.  
Written in AutoHotkey with a clean GUI and customizable settings.

---

## ✨ Features

- Clean and minimal interface
- Customizable keybind
- Adjustable delay
- Toggle ON / OFF system
- Lightweight and easy to use

---

## 📥 Requirements

### 1️⃣ Install AutoHotkey

Download **AutoHotkey v1.1 (Unicode 64-bit)** from the official website:

https://www.autohotkey.com/

⚠ Make sure you install **v1.1**, NOT v2.

---

## 🚀 Installation

1. Install AutoHotkey v1.1 (Unicode 64-bit)
2. Download `MacroDH.ahk` from this repository
3. Open the file using AutoHotkey

---

## ⚙️ How To Use

1. Launch the macro  
2. Click **Set Keybind** and choose your preferred key  
3. Set `Delay = 0`  
4. Click **Save**  
5. Press your selected key to toggle the macro ON/OFF
